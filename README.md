# **Commit: SetUp**

## Objetivo: Copia el proyecto

- Descargar de github:

```bash
git clone https://github.com/Aletheios42/TodoList_Web
```

- Cambiar a la version de inicio

```bash
git checkout <hash_commit>
```

- Instala las dependecias del proyecto.

```npm
npm install
```

- Inicia el server local

```npm
npm start
```
